# CodSoft Data Science Internship Projects

This repository contains all the projects completed during my Data Science Virtual Internship at [CodSoft]([https://www.codsoft.in/]).  Each project is a unique implementation of various AI techniques and serves as a testament to the skills and knowledge gained during the internship.

Duration: 20 March 2026 – 20 April 2026  
Focus: Building end-to-end Machine Learning systems with practical applications

---

## Projects Overview

### 1. Movie Rating Prediction System
A machine learning system that predicts IMDb-style movie ratings based on features such as genre, director, and cast.

**Key Features:**
- Advanced feature engineering (director and actor-based influence)
- Target encoding with smoothing
- Multiple regression models with comparison
- Random Forest as the final model
- Desktop application using Tkinter

**Tech Stack:**  
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Tkinter

---

### 2. Advertising Sales Prediction Dashboard
A regression-based system to predict sales using advertising spend across multiple channels.

**Key Features:**
- Multiple models (Linear, Ridge, Lasso, ElasticNet, Random Forest, SVR)
- Pipeline-based preprocessing
- Model evaluation using MAE, RMSE, R², and cross-validation
- Interactive dashboard for visualization and prediction

**Tech Stack:**  
Python, Scikit-learn, Matplotlib, Seaborn, Tkinter

---

### 3. Credit Card Fraud Detection System
A classification system to detect fraudulent transactions using imbalanced data.

**Key Features:**
- Handling class imbalance using SMOTE
- Hyperparameter tuning using GridSearchCV
- Evaluation using ROC-AUC and Precision-Recall AUC
- Confusion matrix and classification report visualization
- Real-time transaction simulator

**Tech Stack:**  
Python, Scikit-learn, Imbalanced-learn, Matplotlib, Seaborn, Tkinter
